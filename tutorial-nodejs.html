Building a REST API with Node.js

Step-by-step guide to create a simple REST API using Express.

1. Initialize project

npm init -y
npm install express

2. Example server

const express = require('express');
const app = express();
app.get('/ping', (req, res) => res.json({pong:true}));
app.listen(3000);
