This is a Bootstrap-based 10-page personal developer website template built for Abdellahi kah.
It is ready to be edited locally (VS Code) and deployed to GitHub Pages.

## How to use locally
1. Unzip the project and open the folder in VS Code.
2. Start a simple local server (recommended) such as:
   - Python 3: `python -m http.server 8000`
   - Or use the Live Server extension in VS Code.
3. Open http://localhost:8000/index.html in your browser.

## Deploy to GitHub Pages
1. Create a new repository on GitHub and push these files to `main`.
2. In repo Settings -> Pages, set source to `main` branch / root.
3. GitHub will publish the site at `https://<username>.github.io/<repo>/`.

Files included:
- index.html, about.html, projects.html, skills.html, tutorials.html, blog.html, resume.html, contact.html
- tutorial-nodejs.html, tutorial-react.html, tutorial-flask.html
- css/styles.css, js/main.js
- docs/resume.pdf (placeholder), images placeholders
